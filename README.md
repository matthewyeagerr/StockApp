# StockApp
Stock Portfolio Tracker
This is a basic iOS app built with SwiftUI that lets users manually track a personal stock portfolio.

The app allows you to:

Set a starting cash balance

Buy stocks by entering a symbol, number of shares, and purchase price

Track your cash and total portfolio value

Sell shares from your existing positions

This project is intended as a starter app to understand portfolio management logic and SwiftUI basics. All data is stored locally using UserDefaults.

Currently, stock prices are entered manually. Work is in progress to integrate a stock API for real-time price updates.
